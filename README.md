# Proyecto: Juego de la Serpiente (Snake Game)
## Arquitectura: MVC (Modelo-Vista-Controlador)

### Descripción
Implementación del juego Snake siguiendo los principios de diseño arquitectónico propuestos en la Fase 1.

### Componentes:
- **Modelo:** `SnakeModel` - Gestiona la lógica de datos y colisiones.
- **Vista:** `SnakeView` - Renderizado gráfico usando Turtle.
- **Controlador:** `SnakeController` - Gestión de eventos y bucle de juego.
